1 Introduction

There is no 3.0 kit released officially. But all the components that make up the kit (refer DPR page 11) are achieved independently and tracked by a repository on github.

2 Auto Deployment 3.0

Release 3.0 supports multiple operating systems for deployment of applications. Until this release ADS supported multiple flavors of Ubuntu. With this release, deployment of applications is supported on virtual machines running Centos operating system. For more details refer to the release notes of the OVPL 3.0 release.

3 Web 3.0 Compliance

The release contains the documentation and software to port labs to Open edX platform. Once a lab is ported to Open edX platform, the lab becomes responsive. For more details refer to the release notes of the Web Compliance 3.0 release.

4 Analytics 3.0

The analytics service is developed that exposes a set of API to display usage and hits of Virtual Labs. This server stores usage information from the Open edX platform. For more details refer to the release notes of the Analytics release.

5 Secutiy Framework 3.0

All labs at integration levels 5 & 6 are deployed on AWS cloud. The cloud is designed in such a way that the virtual machines hosting the labs are not accessible to the outside world. A layer of security is built on top of these VMs such that the traffic is directed through a single machine called router.

6 Persistence API 3.0

Over 90 labs are ported to Open edX platform. The platform provides persistence where the user in a new session is moved to the same section that he/she has accessed in the previous session. The procedure to setup the platform and port virtual labs to the platforms is part of a repository on github. For more details refer to the release notes of the Persistence 3.0 release.

7 Performance 3.0

As part of the performance improvement of the platform and therefore the Virtual Labs, Central Platform Engineering has automated the creation of the cluster from a configuration. The CPE team has also tabulated the performance metrics for different configurations - CPU, RAM, load etc., by designing a series of performance tests.

8 Packaging 3.0

College Cloud edition of Virtual Labs is released. This edition contains 73 labs. For more details refer to the release notes of the Packaging 3.0 release.
